-------------------概要---------------------
このプログラムはUnityのオブジェクトのTransformの値をCsvファイルに出力するプログラムです。
AddElementをオブジェクトに入れることで追加したい情報を入れる事ができます。

-------------------使い方-------------------
TagTransformExporter
上のウィンドウのToolからTagTransformExporterが使えます。
Tool/TagTransformExporter

出力したいオブジェクト達のタグを指定しexportを押すと出力できます。

AddElement
Parameters : 追加したい情報の数
Parameter : 変数の数
value : 値

StringParams : 追加したい要素の数
Size  : 文字数
Value : 文字

-------------------注意---------------------
開いているファイルを上書き保存するとエラーが起きるので注意してください
